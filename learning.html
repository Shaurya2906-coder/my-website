Operators and Expressions in Python

1. What is an Operator?

Operators are special symbols that perform operations on variables and values.

a = 10
b = 5
print(a + b)   # + is the operator, adds 10 and 5 ‚Üí 15

2. What is an Expression?

An expression is a combination of variables, operators, and values that Python can evaluate and produce a result.

(3 + 4) * 5  # Python calculates it to get 35

Python Inputs and Outputs

1. Input in Python

Python allows you to take input from the user using the input() function.

name = input("Enter your name: ")
print("Hello", name)

üîπ Key Points:

  • input() always returns a string.
  • You can convert the input to other types like int, float, etc.:
age = int(input("Enter your age: "))  # Converts the string to integer

2. Output in Python

Python uses the print() function to display output to the screen.

print("Welcome to Python!")

üîπ Printing Multiple Values:

a = 5
b = 10
print("Sum is", a + b)  # Output: Sum is 15

üîπ Formatted Output (f-strings ‚Äì Python 3.6+):

name = "Shaurya"
age = 18
print(f"My name is {name} and I am {age} years old.")

üîπ sep and end Parameters:

  • sep: separates multiple values (default is space)
  • end: specifies what to print at the end (default is newline \n)
print("Hello", "World", sep="-", end="!!\n")  # Output: Hello-World!!

3. Reading Multiple Inputs

From the same line:

x, y = input("Enter two numbers: ").split()
print(x, y)

You can also convert them:

x, y = map(int, input("Enter two numbers: ").split())

Example Program:

name = input("What is your name? ")
marks = float(input("Enter your marks: "))
print(f"{name} scored {marks} marks in the exam.")

Comments and Documentation

1. What Are Comments?

Comments are lines in the code that are ignored by the Python interpreter. They are used to explain the code, make it readable, and help future developers (including yourself) understand what the code does.

‚úÖ Single-Line Comment

Use # for single-line comments:

# This is a single-line comment
print("Hello")  # This prints Hello

2. Multi-Line Comments (Documentation Strings)

You can use triple quotes (''' or """) to write multi-line comments or docstrings.

"""
This is a multi-line comment
spanning multiple lines.
"""

However, in Python, triple quotes are mainly used for docstrings, not for general comments.

3. Docstrings (Documentation Strings)

Docstrings are special multi-line strings used to describe functions, classes, or modules. They appear immediately after the def or class line.

def add(a, b):
    """
    This function adds two numbers.
    Parameters: a, b
    Returns: sum of a and b
    """
    return a + b

üß™ Accessing Docstrings:

You can access them using the .__doc__ attribute:

print(add.__doc__)

‚úÖ Best Practices

  • Use comments to explain why something is done, not what (which should be clear from the code).
  • Use docstrings to describe the purpose, parameters, and return values of functions and classes.

Explore Python Numbers and Strings

Welcome to the Data Types section. Here, you will learn about Python's Numbers and Strings in detail.

1. Python Numbers

Python supports different types of numeric data:

✅ Types of Numbers

  • int – Integer numbers (e.g. 5, -10, 200)
  • float – Decimal numbers (e.g. 3.14, -0.5, 2.0)
  • complex – Complex numbers (e.g. 2 + 3j)

🔹 Example:

a = 10        # int
b = 3.5       # float
c = 2 + 3j    # complex

✅ Basic Operations

print(5 + 3)    # Addition
print(5 - 3)    # Subtraction
print(5 * 3)    # Multiplication
print(5 / 2)    # Division (float)
print(5 // 2)   # Floor division
print(5 % 2)    # Modulus
print(2 ** 3)   # Exponentiation

2. Python Strings

A string is a sequence of characters enclosed in quotes: ' ', " ", ''' ''', or """ """.

name = "Shaurya"

✅ String Operations:

s = "Hello"
print(len(s))         # Length of the string
print(s.upper())      # HELLO
print(s.lower())      # hello
print(s[0])           # H (Indexing)
print(s[1:4])         # ell (Slicing)

✅ String Concatenation and Repetition

print("Hello" + " World")   # Concatenation
print("Hi" * 3)             # Repetition

✅ f-strings (String Formatting)

name = "Shaurya"
age = 18
print(f"My name is {name} and I am {age} years old.")

🚫 Important Note

Strings are immutable: You cannot change a character in a string like s[0] = 'X'.

Lists and Tuples

✅ What is a List?

A list in Python is an ordered, mutable collection of elements. Lists can contain elements of different data types like integers, strings, floats, etc.

📌 Syntax:

my_list = [10, "apple", 3.14, True]

📌 Common List Operations:

my_list.append("banana")     # Add item to end
my_list[0] = 20              # Modify item
del my_list[1]               # Delete item
print(len(my_list))          # Length of list
print(my_list[0])            # Access by index

📌 Useful List Methods:

  • append(): Adds an item at the end.
  • insert(index, item): Adds item at a specific index.
  • remove(item): Removes the first occurrence.
  • sort(), reverse(): Sorts or reverses the list.
  • pop(): Removes and returns the last item.

✅ What is a Tuple?

A tuple is also an ordered collection of elements, but it is immutable (you can't modify it after creation).

📌 Syntax:

my_tuple = (1, "banana", 4.5)

📌 Tuple with One Element:

t = (5,)  # Must include comma to make it a tuple

📌 Accessing Tuple Elements:

print(my_tuple[1])  # Access like a list

🔁 List vs Tuple Comparison

Feature List ([]) Tuple (())
Mutable Yes No
Performance Slower Faster
Syntax [ ] ( )
Use Case Changing data Fixed, constant data

Dictionaries and Sets

🔷 1. Dictionaries in Python

A dictionary is an unordered, mutable collection of key-value pairs. Each key must be unique and immutable (like strings or numbers), but values can be any type.

✅ Syntax:

student = {
    "name": "Shaurya",
    "age": 18,
    "grade": "A"
}

✅ Accessing Elements:

print(student["name"])        # Output: Shaurya

✅ Common Methods:

  • get(key): Returns value or None
  • keys(): Returns all keys
  • values(): Returns all values
  • items(): Returns key-value pairs
  • update(): Updates dictionary
  • pop(key): Removes item by key

✅ Example:

student["age"] = 19
student["city"] = "Delhi"

🟢 2. Sets in Python

A set is an unordered, mutable, and unindexed collection of unique elements.

✅ Syntax:

fruits = {"apple", "banana", "cherry"}

✅ Key Features:

  • No duplicate values
  • No indexing
  • Fast membership testing (in)

✅ Common Set Operations:

fruits.add("orange")         # Add element
fruits.remove("banana")      # Remove element
"apple" in fruits            # Membership test

✅ Set Operations:

A = {1, 2, 3}
B = {3, 4, 5}

print(A | B)  # Union → {1, 2, 3, 4, 5}
print(A & B)  # Intersection → {3}
print(A - B)  # Difference → {1, 2}

Type Conversion

✅ What is Type Conversion?

Type Conversion refers to changing the data type of a variable or value into another data type.

There are two types of conversions in Python:

🔹 1. Implicit Type Conversion (Automatic)

Python automatically converts one data type to another without user involvement when no data loss occurs.

📌 Example:
x = 5        # int
y = 2.0      # float
z = x + y    # int + float → float
print(z)     # Output: 7.0
# Here, Python converts x from int to float to match the type of y.

🔹 2. Explicit Type Conversion (Type Casting)

The programmer manually converts the data type using built-in functions like:

FunctionDescription
int()Converts to integer
float()Converts to float
str()Converts to string
list()Converts to list
tuple()Converts to tuple
bool()Converts to Boolean
📌 Examples:
a = "123"
b = int(a)        # String to int

c = 3.99
d = int(c)        # Float to int → 3 (decimal dropped)

e = str(100)      # Int to string

⚠️ Notes:

  • Only compatible types can be converted.
  • Trying to convert an invalid string to a number (e.g., "abc" to int) will raise a ValueError.

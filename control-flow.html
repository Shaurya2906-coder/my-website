1. if-else Statements

‚úÖ Purpose:

Used for decision-making in Python. Executes blocks of code depending on conditions.

üî∑ Syntax:

    # code block
elif another_condition:
    # another block
else:
    # fallback block

üî∑ Example:

    print("Eligible to vote")
else:
    print("Not eligible")

2. Loops: for and while

‚úÖ for Loop:

Used to iterate over a sequence like a list, tuple, or string.

for i in range(3):
    print(i)

‚úÖ while Loop:

Repeats while a condition is True.

i = 0
while i < 3:
    print(i)
    i += 1

3. break and continue

‚úÖ break:

Used to exit a loop prematurely.

for i in range(5):
    if i == 3:
        break
    print(i)

‚úÖ continue:

Skips the current iteration and goes to the next.

for i in range(5):
    if i == 3:
        continue
    print(i)

4. match Statements (Python 3.10+)

‚úÖ Purpose:

match is similar to switch-case in other languages, used to compare patterns.

üî∑ Syntax:

    case 1:
        print("One")
    case 2:
        print("Two")
    case _:
        print("Default")
